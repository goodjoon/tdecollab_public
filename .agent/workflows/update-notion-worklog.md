# Notion 을 통한 작업 일지 업데이트
- Notion 의 프로젝트 페이지 : https://www.notion.so/goodjoon/PROJECT-TDE-Collab-306e7b6c217a804fb897dfaa43e2b88c?source=copy_link
- Agent 가 작업을 한 경우 "작업일지" 제목 하위에 "[<날짜>] <오늘의 작업명 제목>" 이름으로 페이지를 만든다. 가장 최근의 작업이 가장 위에 오도록 정렬
- 기존에 오늘 날짜의 작업일지 페이지가 있는 경우, 작업 내용을 업데이트 한다
